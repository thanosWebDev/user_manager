# User Managment - test code app

This Web App was created as a code test project.
It uses create-react-app.
The app lists users info. Users can be edited, deleted or add a new one. It is possible to filter and search. It also includes pagination.

### How to run the App

To run the app locally first make sure you have npm & node installed, then follow this steps:

  - Clone the project from github:  `git clone https://github.com/thanosWebDev/user_manager.git`
  - Change directory into the project: `cd user_manager`
  - Install dependencies: `npm install`
  - Start the server: `npm start`
  - Open your browser to: `http://localhost:3000`
